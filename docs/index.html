FileShareToolkit Dashboard

This GitHub Pages placeholder is safe to publish because it does not include audit data.

To publish the real central dashboard here, generate it into docs/index.html:

.\Invoke-FileShareToolkit_Portable_v3_2.ps1 `
  -GenerateCentralDashboard `
  -CentralJsonPath \\auditserver\ShareAudit\Json `
  -GitHubPagesPath .\docs

Only publish the generated dashboard when the embedded file share, ACL, user, and risk data is approved for the page audience.
